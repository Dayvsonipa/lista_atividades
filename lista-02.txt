# Lista de Exercícios em C

## Bloco 1 — Variáveis, `printf` e `scanf`

### Objetivo do bloco

Neste bloco, o aluno deverá praticar os primeiros fundamentos da linguagem C:

* criação de variáveis;
* uso dos tipos `int`, `float`, `char` e `char[]`;
* leitura de dados com `scanf`;
* exibição de informações com `printf`;
* organização básica de um programa em C;
* montagem de mensagens simples com dados digitados pelo usuário.

## Orientação para os alunos

Em todos os exercícios deste bloco, use:

* `#include <stdio.h>`;
* função `main`;
* variáveis adequadas para cada tipo de dado;
* `printf` para mostrar mensagens;
* `scanf` para ler dados digitados pelo usuário.

Quando precisar ler textos com espaço, o professor poderá orientar o uso de técnicas específicas posteriormente. Neste primeiro momento, o aluno pode usar nomes simples, sem espaço, para facilitar a prática inicial.

## Orientação para os alunos

Neste bloco, escolha corretamente o tipo de variável:

* use `int` para números inteiros;
* use `float` para valores com casas decimais;
* use `%d` para ler ou imprimir inteiros;
* use `%f` para ler valores reais;
* use `%.2f` para exibir valores com duas casas decimais.

Exemplo:

```c
printf("Valor final: R$ %.2f", valor);
```

Sempre teste o programa com valores diferentes para verificar se o cálculo está funcionando corretamente.

2. Criar uma variável para cada dado.

#include <stdio.h>

int main() {

Atenção aos operadores relacionais:

## Orientação para os alunos

Neste bloco, todos os exercícios devem usar `if` e `else`.

A estrutura básica é:

```c
if (condicao) {
    // comandos se a condição for verdadeira
} else {
    // comandos se a condição for falsa
}
```

Exemplo completo:

```c
#include <stdio.h>

int main() {
    int idade;

    printf("Digite sua idade: ");
    scanf("%d", &idade);

    if (idade >= 18) {
        printf("Maior de idade.");
    } else {
        printf("Menor de idade.");
    }

    return 0;
}
```

### Atenção

Use operadores relacionais corretamente:

* `>` maior que;
* `<` menor que;
* `>=` maior ou igual;
* `<=` menor ou igual;
* `==` igual;
* `!=` diferente.

Use `==` para comparar valores.

Exemplo correto:

```c
if (senha == 1234) {
    printf("Senha correta.");
}
```

Exemplo incorreto:

```c
if (senha = 1234) {
    printf("Senha correta.");
}
```

No exemplo incorreto, o símbolo `=` está atribuindo valor, não comparando.

## Orientação para os alunos

Neste bloco, use `else if` quando existir mais de duas possibilidades.

Estrutura básica:

```c
if (condicao1) {
    printf("Primeira possibilidade");
} else if (condicao2) {
    printf("Segunda possibilidade");
} else if (condicao3) {
    printf("Terceira possibilidade");
} else {
    printf("Nenhuma das anteriores");
}
```

### Exemplo completo

```c
#include <stdio.h>

int main() {
    float nota;

    printf("Digite a nota: ");
    scanf("%f", &nota);

    if (nota < 60) {
        printf("Reprovado");
    } else if (nota < 80) {
        printf("Aprovado");
    } else if (nota < 90) {
        printf("Muito bom");
    } else {
        printf("Excelente");
    }

    return 0;
}
```

### Atenção importante

Ao usar faixas numéricas em sequência, muitas vezes não é necessário repetir tudo.

Exemplo:

```c
if (nota < 60) {
    printf("Reprovado");
} else if (nota < 80) {
    printf("Aprovado");
}
```

No segundo teste, o programa só chega ali se a nota **não for menor que 60**.
Então, automaticamente, ela já é maior ou igual a 60.

Isso ajuda a criar condições mais limpas e organizadas.

## Bloco 7 — Condições Compostas com `&&`, `||` e `!`

### Objetivo do bloco

Neste bloco, o aluno deverá praticar condições compostas usando os operadores lógicos da linguagem C.

Os principais operadores trabalhados serão:

* `&&` — operador E;
* `||` — operador OU;
* `!` — operador NÃO.

Esses operadores permitem criar decisões mais completas.

Exemplo com `&&`:

```c
if (idade >= 18 && idade <= 59) {
    printf("Adulto.");
}
```

Nesse caso, as duas condições precisam ser verdadeiras.

Exemplo com `||`:

```c
if (dia == 6 || dia == 7) {
    printf("Fim de semana.");
}
```

Nesse caso, basta uma das condições ser verdadeira.

Exemplo com `!`:

```c
if (!(senha == 1234)) {
    printf("Senha incorreta.");
}
```

Nesse caso, o operador `!` inverte o resultado da condição.

## Orientação para os alunos

Neste bloco, preste atenção na diferença entre `&&` e `||`.

### Operador `&&`

Use `&&` quando **todas as condições precisam ser verdadeiras**.

Exemplo:

```c
if (media >= 60 && frequencia >= 75) {
    printf("Aluno aprovado.");
}
```

Nesse caso, o aluno só será aprovado se tiver média suficiente **e** frequência suficiente.

---

### Operador `||`

Use `||` quando **basta uma das condições ser verdadeira**.

Exemplo:

```c
if (idade >= 60 || deficiente == 1) {
    printf("Atendimento prioritario.");
}
```

Nesse caso, a pessoa terá atendimento prioritário se for idosa **ou** se tiver deficiência.

---

### Operador `!`

Use `!` para inverter uma condição.

Exemplo:

```c
if (!(senha == 1234)) {
    printf("Senha incorreta.");
}
```

Isso significa:

`Se a senha NÃO for igual a 1234.`

Também poderia ser escrito assim:

```c
if (senha != 1234) {
    printf("Senha incorreta.");
}
```

---

## Dica importante

Quando uma condição ficar muito grande, use parênteses para deixar o código mais claro.

Exemplo:

```c
if ((idade >= 18 && ingresso == 1) || autorizacao == 1) {
    printf("Entrada permitida.");
}
```

Os parênteses ajudam o programador a entender melhor a lógica da decisão.

## Orientação para os alunos

O `while` geralmente possui três partes importantes:

### 1. Inicialização

É quando criamos a variável antes do laço.

```c
int contador = 1;
```

### 2. Condição

É a regra que mantém o laço funcionando.

```c
while (contador <= 10)
```

### 3. Atualização

É a alteração que faz o laço caminhar para o fim.

```c
contador++;
```

Exemplo completo:

```c
#include <stdio.h>

int main() {
    int contador = 1;

    while (contador <= 10) {
        printf("%d\n", contador);
        contador++;
    }

    return 0;
}
```

---

## Cuidado com loop infinito

Um erro muito comum é esquecer de atualizar a variável de controle.

Exemplo problemático:

```c
int contador = 1;

while (contador <= 10) {
    printf("%d\n", contador);
}
```

Nesse caso, o valor de `contador` nunca muda.
Então a condição `contador <= 10` será sempre verdadeira, criando um loop infinito.

O correto seria:

```c
int contador = 1;

while (contador <= 10) {
    printf("%d\n", contador);
    contador++;
}
```

## Dica final

Antes de escrever um `while`, pergunte:

`O que precisa acontecer para a repetição parar?`

Essa pergunta ajuda a encontrar a condição correta do laço.

* `int contador = 1` cria e inicia o contador;
* `contador <= 10` define até quando o laço repete;
* `contador++` aumenta o contador a cada volta.

---

## Comparação com `while`

Este código com `while`:

```c
int contador = 1;

while (contador <= 10) {
    printf("%d\n", contador);
    contador++;
}
```

Pode ser escrito com `for` assim:

```c
for (int contador = 1; contador <= 10; contador++) {
    printf("%d\n", contador);
}
```

Os dois fazem a mesma coisa, mas o `for` deixa a repetição controlada mais compacta.

---

## Quando usar `for`

Use `for` principalmente quando você já souber quantas vezes a repetição deve acontecer.

Exemplos:

* repetir 10 vezes;
* ler 5 notas;
* mostrar uma tabuada de 1 a 10;
* percorrer números de 1 até 100;
* somar uma sequência com início e fim definidos.

---

## Cuidados importantes

### 1. Não esquecer o ponto e vírgula dentro do `for`

Formato correto:

```c
for (int i = 1; i <= 10; i++)
```

Formato incorreto:

```c
for (int i = 1, i <= 10, i++)
```

No `for`, as três partes são separadas por ponto e vírgula, não por vírgula.

---

### 2. Cuidado com loop infinito

Exemplo perigoso:

```c
for (int i = 1; i <= 10; i--) {
    printf("%d\n", i);
}
```

Aqui o `i` começa em 1 e vai diminuindo.
Como ele nunca ficará maior que 10, a condição `i <= 10` continuará verdadeira por muito tempo.

---

### 3. Cuidado com o limite

Este código mostra de 1 até 10:

```c
for (int i = 1; i <= 10; i++)
```

Este código mostra de 1 até 9:

```c
for (int i = 1; i < 10; i++)
```

A diferença entre `<` e `<=` muda o resultado final.
