# Lista de Exercícios em C

## Bloco 11 — `do while`

### Objetivo do bloco

Neste bloco, o aluno deverá praticar a estrutura de repetição `do while`.

O `do while` é usado quando queremos garantir que um bloco de comandos seja executado pelo menos uma vez, mesmo que a condição seja falsa logo no início.

Estrutura básica:

```c
do {
    // comandos repetidos
} while (condicao);
```

Exemplo:

```c
int opcao;

do {
    printf("1 - Continuar\n");
    printf("0 - Sair\n");
    printf("Escolha uma opcao: ");
    scanf("%d", &opcao);
} while (opcao != 0);
```

Nesse exemplo, o menu aparece pelo menos uma vez e continua repetindo enquanto o usuário não digitar `0`.

## Orientação para os alunos

O `do while` sempre executa o bloco pelo menos uma vez.

Estrutura:

```c
do {
    comandos;
} while (condicao);
```

Atenção: no `do while`, existe ponto e vírgula depois do `while`.

Exemplo correto:

```c
do {
    printf("Digite uma opcao: ");
    scanf("%d", &opcao);
} while (opcao != 0);
```

Exemplo incorreto:

```c
do {
    printf("Digite uma opcao: ");
    scanf("%d", &opcao);
} while (opcao != 0)
```

No exemplo incorreto, faltou o ponto e vírgula no final.

---

## Diferença entre `while` e `do while`

### `while`

Primeiro testa a condição, depois executa.

```c
while (opcao != 0) {
    printf("Executando...");
}
```

Se a condição começar falsa, o bloco pode nunca executar.

---

### `do while`

Primeiro executa, depois testa a condição.

```c
do {
    printf("Executando...");
} while (opcao != 0);
```

Mesmo que a condição seja falsa depois, o bloco executa pelo menos uma vez.

---

## Quando usar `do while`

Use `do while` principalmente em situações como:

* menus;
* validação de entrada;
* jogos que devem começar pelo menos uma vez;
* perguntas do tipo “deseja continuar?”;
* sistemas que repetem até o usuário escolher sair.

---

## Dica final

Uma pergunta que ajuda muito:

`O programa precisa executar pelo menos uma vez antes de testar a condição?`

Se a resposta for sim, o `do while` pode ser uma ótima escolha.

## Conceitos importantes

### Contador

Um contador é uma variável usada para contar alguma coisa.

Exemplo:

```c
aprovados++;
```

Isso significa:

`aprovados = aprovados + 1;`

---

### Acumulador

Um acumulador é uma variável usada para somar valores ao longo do programa.

Exemplo:

```c
total = total + valor;
```

Ou:

```c
total += valor;
```

---

### Média

Para calcular uma média, normalmente usamos:

```c
media = soma / quantidade;
```

Por isso, geralmente precisamos de:

* um acumulador para somar;
* um contador para contar.

## Orientação para os alunos

Antes de começar um exercício com repetição, identifique se você precisará de:

### 1. Contador

Usado para contar ocorrências.

Exemplo:

```c
if (nota >= 60) {
    aprovados++;
}
```

---

### 2. Acumulador

Usado para somar valores.

Exemplo:

```c
somaNotas += nota;
```

---

### 3. Maior valor

Uma forma comum é iniciar o maior valor com o primeiro número digitado.

Exemplo:

```c
if (i == 1) {
    maior = numero;
} else if (numero > maior) {
    maior = numero;
}
```

---

### 4. Menor valor

Também pode ser iniciado com o primeiro valor digitado.

Exemplo:

```c
if (i == 1) {
    menor = numero;
} else if (numero < menor) {
    menor = numero;
}
```

---

### 5. Média

Normalmente exige soma e quantidade.

Exemplo:

```c
media = soma / quantidade;
```

---

## Dica importante

Em exercícios com relatórios, o resultado final geralmente só aparece **depois que o laço termina**.

Exemplo:

```c
for (int i = 1; i <= 10; i++) {
    printf("Digite uma nota: ");
    scanf("%f", &nota);

    soma += nota;
}

media = soma / 10;

printf("Media da turma: %.2f", media);
```

O laço coleta e processa os dados.
Depois do laço, o programa mostra o resumo.

Nesse exemplo:

## Orientação para os alunos

Laços aninhados são repetições dentro de repetições.

Exemplo:

```c
for (int linha = 1; linha <= 3; linha++) {
    for (int coluna = 1; coluna <= 5; coluna++) {
        printf("*");
    }
    printf("\n");
}
```

### Como entender?

Pense em uma sala de aula:

* o laço de fora anda pelas fileiras;
* o laço de dentro anda pelas carteiras de cada fileira.

Ou pense em uma tabela:

* o laço externo controla as linhas;
* o laço interno controla as colunas.

---

## Exemplo comentado

```c
#include <stdio.h>

int main() {
    int linha, coluna;

    for (linha = 1; linha <= 3; linha++) {
        for (coluna = 1; coluna <= 5; coluna++) {
            printf("*");
        }

        printf("\n");
    }

    return 0;
}
```

### Explicação

Quando `linha` vale 1, o laço interno imprime 5 asteriscos.

Depois o programa pula uma linha.

Quando `linha` vale 2, o laço interno imprime mais 5 asteriscos.

Depois pula outra linha.

Quando `linha` vale 3, o laço interno imprime mais 5 asteriscos.

Resultado:

```text
*****
*****
*****
```

---

## Dica importante

Quando aparecer um problema com linhas e colunas, pense imediatamente:

```text
laço externo = linhas
laço interno = colunas
```

Essa ideia será muito útil quando chegarmos em matrizes.

## Orientação para os alunos

Um vetor é uma estrutura que guarda vários valores do mesmo tipo.

Exemplo:

```c
int numeros[5];
```

Esse vetor possui 5 posições:

```text
numeros[0]
numeros[1]
numeros[2]
numeros[3]
numeros[4]
```

---

## Exemplo de preenchimento de vetor

```c
#include <stdio.h>

int main() {
    int numeros[5];
    int i;

    for (i = 0; i < 5; i++) {
        printf("Digite o numero da posicao %d: ", i);
        scanf("%d", &numeros[i]);
    }

    return 0;
}
```

---

## Exemplo para mostrar o vetor

```c
#include <stdio.h>

int main() {
    int numeros[5];
    int i;

    for (i = 0; i < 5; i++) {
        printf("Digite o numero da posicao %d: ", i);
        scanf("%d", &numeros[i]);
    }

    printf("\nValores digitados:\n");

    for (i = 0; i < 5; i++) {
        printf("%d\n", numeros[i]);
    }

    return 0;
}
```

---

## Atenção com os índices

Em C, os vetores começam na posição zero.

Se o vetor tem 5 posições:

```c
int numeros[5];
```

As posições válidas são:

```text
0, 1, 2, 3, 4
```

A posição `5` não pertence ao vetor.

Exemplo incorreto:

```c
numeros[5] = 10;
```

Isso tenta acessar uma posição que não existe.

---

## Dica importante

Quando usamos vetor com `for`, quase sempre a estrutura fica assim:

```c
for (i = 0; i < tamanho; i++) {
    // usar vetor[i]
}
```

Exemplo:

```c
for (i = 0; i < 10; i++) {
    scanf("%d", &numeros[i]);
}
```

Repare que a condição é `i < 10`, e não `i <= 10`.

Isso acontece porque as posições vão de `0` até `9`.

para todas as posições.

## Orientação para os alunos

Uma matriz é parecida com uma tabela.

Exemplo de matriz 3x3:

```text
[0][0] [0][1] [0][2]
[1][0] [1][1] [1][2]
[2][0] [2][1] [2][2]
```

Cada posição possui dois índices:

```c
matriz[linha][coluna]
```

## Exemplo de preenchimento de matriz

```c
#include <stdio.h>

int main() {
    int matriz[3][3];
    int linha, coluna;

    for (linha = 0; linha < 3; linha++) {
        for (coluna = 0; coluna < 3; coluna++) {
            printf("Digite o valor da posicao [%d][%d]: ", linha, coluna);
            scanf("%d", &matriz[linha][coluna]);
        }
    }

    return 0;
}
```

---

## Exemplo para mostrar a matriz

```c
#include <stdio.h>

int main() {
    int matriz[3][3];
    int linha, coluna;

    for (linha = 0; linha < 3; linha++) {
        for (coluna = 0; coluna < 3; coluna++) {
            printf("Digite o valor da posicao [%d][%d]: ", linha, coluna);
            scanf("%d", &matriz[linha][coluna]);
        }
    }

    printf("\nMatriz digitada:\n");

    for (linha = 0; linha < 3; linha++) {
        for (coluna = 0; coluna < 3; coluna++) {
            printf("%d ", matriz[linha][coluna]);
        }
        printf("\n");
    }

    return 0;
}
```

## Atenção aos índices

Em C, os índices começam em zero.

Uma matriz 3x3:

```c
int matriz[3][3];
```

possui linhas:

```text
0, 1, 2
```

e colunas:

```text
0, 1, 2
```

Portanto, a última posição é:

```c
matriz[2][2]
```

Não existe:

```c
matriz[3][3]
```

---

## Como pensar em matrizes

Sempre pense assim:

```text
laço externo = linha
laço interno = coluna
```

Exemplo:

```c
for (linha = 0; linha < 3; linha++) {
    for (coluna = 0; coluna < 3; coluna++) {
        scanf("%d", &matriz[linha][coluna]);
    }
}
```

O primeiro `for` controla as linhas.

O segundo `for` controla as colunas.

---

## Diagonal principal

Em uma matriz quadrada, a diagonal principal acontece quando:

```c
linha == coluna
```

Exemplo em matriz 3x3:

```text
X . .
. X .
. . X
```

---

## Diagonal secundária

Em uma matriz quadrada 3x3, a diagonal secundária acontece quando:

```c
linha + coluna == 2
```

Em uma matriz de tamanho `n`, a regra geral é:

```c
linha + coluna == n - 1
```

Exemplo:

```text
. . X
. X .
X . .
```

* boletim de alunos;
* mapa de sala;
* tabuleiro de jogo;
* controle de estoque por setor;
* tabelas de preços;
* imagens digitais;
* jogos da velha;
* mapas e grades.

Quando o problema tiver linhas e colunas, provavelmente uma matriz será uma boa solução.

Exemplo:

```c
char nome[50];
```

Isso significa que a variável `nome` pode armazenar vários caracteres.

## Exemplo com `scanf`

```c
#include <stdio.h>

int main() {
    char nome[50];

    printf("Digite seu primeiro nome: ");
    scanf("%s", nome);

    printf("Nome digitado: %s", nome);

    return 0;
}
```

Esse exemplo funciona bem para nomes sem espaço.

---

## Exemplo com `fgets`

```c
#include <stdio.h>

int main() {
    char nome[100];

    printf("Digite seu nome completo: ");
    fgets(nome, 100, stdin);

    printf("Nome digitado: %s", nome);

    return 0;
}
```

O `fgets` permite ler textos com espaço.

---

## Removendo o `\n` do `fgets`

Quando usamos `fgets`, ele pode guardar a quebra de linha no final da string.

Exemplo para remover:

```c
#include <stdio.h>
#include <string.h>

int main() {
    char nome[100];
    int tamanho;

    printf("Digite seu nome: ");
    fgets(nome, 100, stdin);

    tamanho = strlen(nome);

    if (tamanho > 0 && nome[tamanho - 1] == '\n') {
        nome[tamanho - 1] = '\0';
    }

    printf("Nome sem quebra de linha: %s", nome);

    return 0;
}
```

---

## Comparando strings

Em C, não comparamos strings assim:

```c
if (senha == "1234")
```

Esse formato está incorreto para comparar textos.

O correto é usar `strcmp`:

```c
#include <string.h>

if (strcmp(senha, "1234") == 0) {
    printf("Senha correta.");
}
```

A função `strcmp` retorna `0` quando as duas strings são iguais.

---

## Copiando strings

Para copiar uma string, usamos `strcpy`.

```c
strcpy(destino, origem);
```

Exemplo:

```c
char origem[50] = "C";
char destino[50];

strcpy(destino, origem);
```

---

## Concatenando strings

Para juntar strings, usamos `strcat`.

```c
strcat(texto1, texto2);
```

Exemplo:

```c
char nome[100] = "Maria";
char sobrenome[50] = "Silva";

strcat(nome, " ");
strcat(nome, sobrenome);
```

Resultado:

```text
Maria Silva
```

---

## Percorrendo uma string caractere por caractere

Podemos percorrer uma string usando `for`:

```c
#include <stdio.h>
#include <string.h>

int main() {
    char palavra[50];
    int i;

    printf("Digite uma palavra: ");
    scanf("%s", palavra);

    for (i = 0; i < strlen(palavra); i++) {
        printf("%c\n", palavra[i]);
    }

    return 0;
}
```

---

## Atenção importante

Uma string em C termina com o caractere especial:

```c
'\0'
```

Esse caractere indica o final do texto.

Por isso, se criamos:

```c
char nome[5] = "Joao";
```

A string ocupa:

```text
J o a o \0
```

Ou seja, 4 letras mais o finalizador.

---

## Dica final

Use:

* `scanf("%s", texto)` para palavras sem espaço;
* `fgets(texto, tamanho, stdin)` para frases ou nomes completos;
* `strlen` para contar caracteres;
* `strcmp` para comparar strings;
* `strcpy` para copiar strings;
* `strcat` para juntar strings.

Strings parecem simples, mas exigem atenção porque, em C, texto é tratado como vetor de caracteres.

void mostrarLinha() {

### Função com parâmetro

```c
void mostrarDobro(int numero) {

```

Chamada:

```c
mostrarDobro(5);
```

### Função para organizar código

void mostrarMenu() {

## Orientação para os alunos

Funções ajudam a organizar o código.

Sem funções, tudo fica dentro da `main`, e o programa pode ficar grande e confuso.

Com funções, cada parte do programa recebe uma responsabilidade.

---

## Estrutura básica de uma função `void`

```c
void nomeDaFuncao() {
    printf("Mensagem da funcao.\n");
}
```

Para usar a função, chamamos pelo nome:

```c
nomeDaFuncao();
```

---

## Exemplo completo

```c
#include <stdio.h>

void mostrarLinha() {
    printf("------------------------------\n");
}

void mostrarTitulo() {
    mostrarLinha();
    printf("       SISTEMA ESCOLAR\n");
    mostrarLinha();
}

int main() {
    mostrarTitulo();

    printf("Programa iniciado.\n");

    return 0;
}
```

---

## Função com parâmetro

Um parâmetro é uma informação que a função recebe para trabalhar.

Exemplo:

```c
void mostrarDobro(int numero) {
    printf("Dobro: %d\n", numero * 2);
}
```

Chamada:

```c
mostrarDobro(8);
```

Saída:

```text
Dobro: 16
```

---

## Função com mais de um parâmetro

```c
void mostrarSoma(int a, int b) {
    printf("Soma: %d\n", a + b);
}
```

Chamada:

```c
mostrarSoma(10, 5);
```

Saída:

```text
Soma: 15
```

---

## Atenção importante

Quando a função é `void`, ela não devolve um valor para quem chamou.

Ela apenas executa uma tarefa.

Exemplo:

```c
void mostrarMensagem() {
    printf("Ola!");
}
```

Essa função mostra uma mensagem, mas não retorna nada.

---

## Organização recomendada

Uma boa organização inicial é:

```c
#include <stdio.h>

void minhaFuncao() {
    // comandos da funcao
}

int main() {
    minhaFuncao();

    return 0;
}
```

---

## Dica final

Sempre que você perceber que um trecho de código pode ser usado mais de uma vez, pense:

`Isso poderia virar uma função?`

Funções deixam o código mais limpo, organizado, reutilizável e fácil de corrigir.



# Lista de Exercícios em C

## Bloco 18 — Funções com Retorno e Parâmetros

### Objetivo do bloco

Neste bloco, o aluno deverá praticar funções que recebem valores por parâmetro, realizam algum processamento e retornam um resultado usando `return`.

A ideia é fazer o aluno entender a diferença entre:

* função que apenas executa uma ação;
* função que calcula e devolve um valor;
* função `void`;
* função `int`;
* função `float`;
* função `char`;
* função que retorna resultado lógico usando `int`.

---

## Conceitos importantes

### Função sem retorno

Uma função `void` executa uma tarefa, mas não devolve valor.

```c
void mostrarMensagem() {
    printf("Ola!");
}
```

---

### Função com retorno

Uma função com retorno devolve um valor para quem chamou.

Exemplo:

```c
int somar(int a, int b) {
    return a + b;
}
```

Uso na `main`:

```c
int resultado;

resultado = somar(10, 5);

printf("Resultado: %d", resultado);
```

---

### Tipos comuns de retorno

```c
int     // retorna número inteiro
float   // retorna número real
char    // retorna caractere
```

Exemplos:

```c
int dobro(int numero) {
    return numero * 2;
}
```

```c
float media(float n1, float n2) {
    return (n1 + n2) / 2;
}
```

```c
char primeiraLetra() {
    return 'A';
}
```

## Orientação para os alunos

Uma função com retorno tem esta estrutura:

```c
tipo nomeDaFuncao(parametros) {
    return valor;
}
```

Exemplo:

```c
int somar(int a, int b) {
    return a + b;
}
```

Na `main`, podemos guardar o resultado em uma variável:

```c
int resultado;

resultado = somar(10, 5);

printf("Resultado: %d", resultado);
```

---

## Diferença entre `void` e função com retorno

### Função `void`

A função executa algo, mas não devolve valor.

```c
void mostrarDobro(int numero) {
    printf("Dobro: %d", numero * 2);
}
```

Uso:

```c
mostrarDobro(5);
```

---

### Função com retorno

A função calcula e devolve um valor.

```c
int calcularDobro(int numero) {
    return numero * 2;
}
```

Uso:

```c
int resultado;

resultado = calcularDobro(5);

printf("Dobro: %d", resultado);
```

---

## Por que usar retorno?

Porque a função fica mais flexível.

Exemplo:

```c
float calcularMedia(float n1, float n2) {
    return (n1 + n2) / 2;
}
```

Depois, a `main` pode decidir o que fazer com essa média:

```c
media = calcularMedia(80, 70);

if (media >= 60) {
    printf("Aprovado");
} else {
    printf("Reprovado");
}
```

A função apenas calcula.
A decisão pode ficar em outra parte do programa.

---

## Exemplo completo

```c
#include <stdio.h>

float calcularMedia(float n1, float n2) {
    return (n1 + n2) / 2;
}

int foiAprovado(float media) {
    if (media >= 60) {
        return 1;
    } else {
        return 0;
    }
}

int main() {
    float nota1, nota2, media;

    printf("Digite a primeira nota: ");
    scanf("%f", &nota1);

    printf("Digite a segunda nota: ");
    scanf("%f", &nota2);

    media = calcularMedia(nota1, nota2);

    printf("Media: %.2f\n", media);

    if (foiAprovado(media) == 1) {
        printf("Aluno aprovado.");
    } else {
        printf("Aluno reprovado.");
    }

    return 0;
}
```

---

## Dica importante

Quando uma função retorna `1` ou `0`, ela pode ser usada como uma pergunta.

Exemplo:

```c
if (ehPar(numero) == 1) {
    printf("Numero par.");
}
```

Também é comum escrever:

```c
if (ehPar(numero)) {
    printf("Numero par.");
}
```

Porque em C:

* `0` significa falso;
* qualquer valor diferente de `0` significa verdadeiro.

---

## Cuidados importantes

### 1. O tipo do retorno precisa combinar com o valor retornado

Exemplo correto:

```c
float calcularMetade(int numero) {
    return numero / 2.0;
}
```

Exemplo problemático:

```c
int calcularMetade(int numero) {
    return numero / 2.0;
}
```

Nesse segundo caso, a função retorna `int`, então a parte decimal pode ser perdida.

---

### 2. Toda função com retorno deve retornar algo

Exemplo incorreto:

```c
int somar(int a, int b) {
    int resultado = a + b;
}
```

Faltou o `return`.

Exemplo correto:

```c
int somar(int a, int b) {
    int resultado = a + b;
    return resultado;
}
```

---

### 3. A função deve ter uma responsabilidade clara

Evite criar uma função que faz tudo ao mesmo tempo.

Melhor:

```c
float calcularMedia(float n1, float n2)
```

Pior:

```c
void lerNotasCalcularMediaMostrarResultadoVerificarAprovacao()
```

Funções menores deixam o programa mais organizado.

---

## Dica final

Ao criar uma função, pergunte:

1. Qual é o nome da função?
2. Que dados ela precisa receber?
3. Que tipo de valor ela precisa devolver?
4. Qual será o `return`?

Se conseguir responder essas quatro perguntas, a função ficará muito mais fácil de montar.

    float frequencia;
};

## Exemplo completo com uma struct

```c

struct Aluno {
    char nome[50];
    int idade;
    float media;
};

int main() {
    struct Aluno aluno;

    printf("Digite o nome do aluno: ");
    scanf("%s", aluno.nome);

    printf("Digite a idade: ");
    scanf("%d", &aluno.idade);

    printf("Digite a media: ");
    scanf("%f", &aluno.media);

    printf("\nDados do aluno:\n");
    printf("Nome: %s\n", aluno.nome);
    printf("Idade: %d\n", aluno.idade);
    printf("Media: %.2f\n", aluno.media);

```

## Exemplo com vetor de struct

```c
#include <stdio.h>

struct Produto {
    char nome[50];
    float preco;
    int quantidade;
};

int main() {
    struct Produto produtos[3];
    int i;

    for (i = 0; i < 3; i++) {
        printf("Digite o nome do produto %d: ", i + 1);
        scanf("%s", produtos[i].nome);

        printf("Digite o preco: ");
        scanf("%f", &produtos[i].preco);

        printf("Digite a quantidade: ");
        scanf("%d", &produtos[i].quantidade);
    }

    printf("\nProdutos cadastrados:\n");

    for (i = 0; i < 3; i++) {
        printf("\nProduto %d\n", i + 1);
        printf("Nome: %s\n", produtos[i].nome);
        printf("Preco: R$ %.2f\n", produtos[i].preco);
        printf("Quantidade: %d\n", produtos[i].quantidade);
    }

    return 0;
}
```

## Acessando campos da struct

Para acessar um campo, usamos ponto:

```c
produto.preco
aluno.nome
funcionario.salario
```

Exemplo:

```c
printf("Preco: %.2f", produto.preco);
```

---

## Struct com função

Também podemos passar uma `struct` para uma função:

```c
void mostrarProduto(struct Produto produto) {
    printf("Nome: %s\n", produto.nome);
    printf("Preco: %.2f\n", produto.preco);
    printf("Quantidade: %d\n", produto.quantidade);
}
```

Chamada:

```c
mostrarProduto(produto1);
```

---

## Dica importante

Use `struct` quando vários dados pertencem ao mesmo objeto.

Exemplos:

### Aluno

* nome;
* idade;
* média;
* frequência.

### Produto

* código;
* nome;
* preço;
* estoque.

### Funcionário

* nome;
* cargo;
* salário.

### Livro

* título;
* autor;
* ano;
* quantidade disponível.

---

## Dica final

Sem `struct`, os dados ficam espalhados:

```c
char nome[50];
float preco;
int quantidade;
```

Com `struct`, os dados ficam organizados:

```c
struct Produto {
    char nome[50];
    float preco;
    int quantidade;
};
```

Isso deixa o código mais próximo de sistemas reais e muito mais fácil de entender, manter e expandir.

## Estratégia recomendada

Para projetos maiores, não tente fazer tudo de uma vez.

Siga esta ordem:

### 1. Crie a `struct`

Exemplo:

```c
struct Aluno {
    int matricula;
    char nome[100];
    int idade;
    float nota1;
    float nota2;
    float nota3;
    float media;
    float frequencia;
};
```

### 2. Crie o vetor

```c
struct Aluno alunos[10];
```

### 3. Crie o menu

```c
do {
    printf("1 - Cadastrar\n");
    printf("2 - Listar\n");
    printf("0 - Sair\n");
    scanf("%d", &opcao);
} while (opcao != 0);
```

### 4. Faça uma função por vez

Não tente resolver o sistema inteiro de uma vez.

Comece por:

* cadastrar;
* listar;
* buscar;
* calcular;
* gerar relatório.

### 5. Teste a cada etapa
